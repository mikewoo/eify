# 行为准则

## 我们的承诺

为了营造一个开放和友好的社区环境，作为本项目的贡献者和维护者，我们承诺尊重所有参与者——无论其经验水平、性别、性取向、残疾状况、外貌、种族、年龄或宗教信仰。

## 我们的准则

### 鼓励的行为

- 友好、包容、耐心地对待他人
- 尊重不同的观点和经验
- 建设性地接受批评
- 关注对社区最有利的事情
- 对其他社区成员抱有同理心

### 禁止的行为

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注
- 挑衅、侮辱或贬低他人的评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，发布他人的私密信息（如地址、邮箱等）
- 其他在专业环境中被认为不当的行为

## 我们的责任

项目维护者有责任明确和执行可接受行为的标准，并对任何不可接受的行为采取适当和公正的纠正措施。

维护者有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 和其他贡献，并有权暂时或永久禁止他们认为有不当、威胁、冒犯或有害行为的贡献者。

## 范围

本行为准则适用于项目空间和公共空间。当个人代表项目或其社区时，同样适用。

## 报告

如果你遇到滥用、骚扰或其他不可接受的行为，请通过创建 Issue 或直接联系项目维护者进行报告。所有投诉将被审查和调查，并将产生被认为必要且适合情况的回应。
